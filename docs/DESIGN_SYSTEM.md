# AudiesChess Design System

Version: 0.1
Status: Core

---

# Design Philosophy

AudiesChess is clean, modern, premium and AI-first.

The interface should feel calm, focused and professional.

Avoid visual clutter.

Every component should have a clear purpose.

# Shadows

Avoid heavy shadows.

Use a single icon library across the project.

Icons should be simple and consistent.

Avoid mixing different icon styles.
